### Prereqs:
- make sure you have python installed
- fork the repo and git clone it into a folder
- inside that folder, please make a python virual enviroment
- change python interperter to the virtual enviroment one
- activate the environment in your terminal
- install flask using pip
- install flask-mysql using pip
- download XXAMPP 
- start the Apache and MySQL
-Press Admin in the MySQL section which should redirect you to phpmyadmin
- here make a database called "car_rental_db"
- import the sql file from databaseSQL folder in this repo to the database
